# DIGO_EXE — 總覽

> 一款**兩人對戰回合制遊戲**的設計工作區(**數位、之後由 Claude 寫成程式**)。大方向:把《洛克人EXE》的格子晶片戰改成回合制,並借《聖劍同盟》的「卡片自帶移動力」。
>
> **2026-07-03 框架重整**:設計基準改為「確立項與發想分離」——已確立的骨架收在 [[0_核心確立項]](唯一規則權威),其餘一切是發想、不是死規則;舊 v1 規則稿整批歸檔 `9_系統/_歷史/`。

## 核心

- **核心樂趣三支柱**:選擇、資源分配(一回合內行動有限)、猜想對方動作與競爭。
- **勝利條件**:把對方 HP 扣到 0。
