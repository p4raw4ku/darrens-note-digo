# 發想:吃中線

**來源**:舊 v1 規則稿的招牌之一(完整條文見 `9_系統/_歷史/1_核心規則.md` §6:動態中線、推進奪行、三重鎖、冷卻、逆境補償、擠出規則)。

## 描述

打出吃中線效果,把中線往對手方向推進一行:對手最前一行併入你的領地。舊稿配套:一次一行、累計上限兩行、可推回、冷卻、弱勢方推回步幅加倍。

## 價值

- 領地攻防是走位的進攻性用途(不只閃避)。
- 給「搶地盤」型晶片與領航員能力一個家。
- 推線後攻擊發起位置前移,射程經濟改變——空間壓力的來源。

## 已知的坑(審查備查,回鍋必修)

1. **安全屋**:領地變深但射程上限固定 → 推滿後勝方最深兩行永遠打不到,縮在裡面不互動反而穩贏(平手裁定若「比領地」更是直接獎勵停止互動)。這不是調數字能解的:射程加大則開局全場互打、前後走位死;射程不變則推線後必有死角。
2. **替代案(視窗滑移)**:把「我的地變大」改成「整個戰場前移」——推進一行、自己最後一行同步封閉,雙方可用縱深恆為三。安全屋結構性消失、射程校準永遠成立、推線甜頭保留(對手可退空間變少、你的開火位置前移)。
3. 冷卻的計時主體要寫明(「間隔一回合」用絕對回合讀,對手的回合天然填掉間隔、鎖形同虛設)。
4. 兩個九宮格的新場地下,「中線」的定義本身要等 [[題5_幾何條款]] (跨盤結構)拍板才有意義。
