# DIGO_EXE — 角色圖鑑

手繪角色圖總表，分**領航員**與**病毒**兩類（2026-07-10 拍板：病毒暫無規則類別之分、先當角色屬性）。此刻只收圖與分類——每隻未來將補「**規則區**」（三張基礎晶片／固有能力／HP 與防禦數值），等 [[題8_質變與領航員能力]] 的質變形態與題4 數值拍板後逐隻填（規則唯一權威仍是 `1_確立項/`）。首張已定案：洛克人固有能力＝可啟動其他領航員晶片的質變能力。
