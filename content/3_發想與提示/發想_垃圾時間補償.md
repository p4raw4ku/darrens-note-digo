# 發想:垃圾時間補償

**來源**:2026-07-10 user 拍 3對3 全滅制時的附帶設計需求——大劣後的垃圾時間要有機制處理或補償。

## 設計邊界

- 補償量級**不能抵銷擊殺的價值**(否則擊殺無獎勵、全滅制被拖長)。
- 觸發要客觀:**掉員**(離散事件、可讀、無爭議);HP 連續落差不觸發,留給題6 資源系統。
- 反向利用檢查已過:全滅制下故意送人頭=幫對手完成三分之一勝利條件,合理量級的補償補不回來,餵人頭不成立。

## 候選(可疊加)

| 案 | 內容 | 評 |
|---|---|---|
| 甲 | 掉員後手牌上限+1(按掉員數疊加、永久) | **骨幹首選**——持續型補償對持續型劣勢(質變出口/換角選項/HP 池的損失是永久的),加在最重要資源上,逐回合複利、幫翻盤不幫拖延 |
| 丙 | 掉員當回合多抽 2 | **急救包**——掉員當輪最危險(對手挾勢追擊、剛損失一個質變體系),一次性補給組得出防線;與甲性質互補 |
| 丁 | 遺志繼承:陣亡者三張基礎晶片對全隊保留質變 | 世界觀糖衣、量級小而永久;補償的正是實際失去的東西(質變出口);對洛克人玩家近乎冗餘;可測可不上 |
| 戊 | 認輸鈕+系統判定提示 | **兜底(非補償)**——真垃圾時間沒有補償救得回來;數位實作成本近零 |
| 乙 | 掉員當回合換兩張 | 建議捨——質不是量、弱於丙,且與後手洗二補償撞型 |

**建議組合:甲+丙+戊**,丁選配。垃圾時間的另一半解法在對面——「優勢方終結得快」靠終局機制(題7),同一枚硬幣兩面、串拍。
