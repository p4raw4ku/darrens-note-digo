# 確立項:領航員

## 已確立

- 領航員有**自身能力**、**自身攻擊**、**HP**。
- 領航員可具備**防禦三機制**(可帶哪幾種、數值未定),機制定義:
  - **閃避**:每點有機率免除一整擊(機率制;確定性替代框架「超額削減」見 [[1_設計約束與審查備查]] 存活條)。
  - **護盾**:回充窗——護盾值抵消傷害,歸 0 後下一擊全額穿透、該擊之後回滿。
  - **力場**:門檻——每擊先減力場值;單次傷害不超過力場值,該擊整個歸零。

## 設計意圖

- 自身攻擊是保底節奏(手上沒好晶片的回合仍有輸出),定價原則參考《洛克人EXE》:基線弱而穩、晶片強而稀缺——否則晶片變可選配件。
- 自身能力是角色差異的主載體;能力設計的兩條教訓(強度別與核心硬度耦合、反制要有成本)見 [[1_設計約束與審查備查]] 備查條與 [[發想_角色破例_衝鋒與繞側]]。

## 未定與掛靠

- 能力的內容、HP 數值 → [[題2_隊伍規模]](1對1 或 3對3 決定能力設計空間)、[[題4_防禦數值]]
- 三防禦的數值帶、閃避採機率或確定性、一名領航員可帶幾種、防禦是否常駐可見 → 題4
- 結算細節(逐擊狀態機)→ [[5_傷害模型]](綁確立項、沿用)
