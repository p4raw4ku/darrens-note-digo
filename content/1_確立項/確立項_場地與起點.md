# 確立項:場地與起點

## 已確立

- 雙方各持一個**九宮格**(3×3),起點各自中央。
- **面對面、鏡像對應**:實體為一張 3 橫排×6 縱列的連續棋盤;雙方各自視角下「我的左排=你的右排」;引擎內部用絕對座標(等價)。
- **座標系(2026-07-13 拍)**:**欄＝A–F**(縱列)、**列＝1/2/3**(橫排、上中下)——溝通與條文一律用此座標;我方 A–C、敵方 D–F。
- **跨盤距離**:沿格直數,過中線無修正;距離與範圍一律從角色所在格起算(錨點鐵則)。
- **移動步法(2026-07-14 拍)**:移動以**上下左右**為單位步、**斜向=兩步**;晶片移動力 N=單位步預算,每一步都過位移硬上限(不得出邊界、不得跨當前領域界線),目的格存在**任一條全程合法路徑**即可移動(路徑判定=實作裁量、引擎以逐步搜尋判可達,待追認)。此條只管「移動」;位移(推/拉)與範圍判定本來就是直向、不受影響。舊佔位池引擎的「八方一步斜走」係層1 起未經拍板的實作假設、已由本條取代(引用佔位池時代走位數據時標註)。
- **領域界線(2026-07-13 升級為逐列)**:每一列各自一條界線、預設全在 C|D;**不可進入對方領域**(特殊效果才破例)。界線可被效果改變(現行接口=佔領詞條),變更後所有領域判定以**當前該列**界線重算。
- **佔領(界線變更詞條,2026-07-13 拍全套)**:單格版(敵前 1 格)與整欄版(敵前縱列)——①**整欄推進鎖(2026-07-15 改綁整欄卡)**:離發動方最近的欄三列全滿後,才可再以「佔領(整欄)」佔下一欄;**「佔領(單格)」不受此限、可延伸**——目標=發動者**自己那一列的前緣下一格**(逐格長、不可跳格;發動者換列後即可延伸新列——移動×佔領聯動,2026-07-15 深夜拍 U4);**整欄卡打在已有走廊格的欄=補滿剩餘格**(U5;站人格照擠壓);②**站人格不變更**:佔領當下站著單位的格不變更歸屬,該單位改吃擠壓傷害 10(**直傷**,2026-07-14 批准:不經閃避/力場/護盾判定——「場地傷害」家族,掛靠見「未定與掛靠」);③敵方最後一欄(F)不可佔;④**時效=逐格回合時鐘(2026-07-15 改制)**:每個佔領格自己一支時鐘、以**回合**計——**暫定 3 回合**、**佔領當回合算第 1 回合**(U6 拍:第 3 個回合結束到期;回合數待掃描,住 [[1_準確立層]];窗要容納佔領→壓進→劍系兌現)。**復原**:到期後由**最遠端格先退**(離發動方最遠;較近格的倒數必早於或等於較遠格、但仍等更遠格復原後才輪到——07-14「最深先退」原則沿用);各格的復原時點=該格上**無佔領方角色**時的**第一個行動前**(站人先等待沿用、錨點=行動);**復原節奏=每列每個行動前至多退一格(2026-07-16 拍)**——更遠格復原後,較近格於**下一個行動前**才退(同瞬間連退不採)。推論(引擎驗收釘住):近格到期前佔更深一格=整列改等深格時鐘、**近格續命**——合法打法,價值與濃度入佔領重估(U16)量測;續命上限=敵方最後一欄不可佔。07-14「回合步」口徑作廢(行動經濟改制連動,見 [[確立項_回合與行動]])。
- **位移硬上限**:任何位移(推/拉/自移)不得使單位移出棋盤邊界、不得使單位跨越**當前**領域界線。例:界線在 C/D 時,不能把對方從 D 拉到 C;若界線已被效果改到 B/C 之間,則 D 拉到 C 合規(對方仍在其領域內)。**逐步鉗制(2026-07-14 拍)**:多格位移逐格推進結算,碰到邊界或界線就停在最後一個合法格——**部分位移合法**、已走的格數保留,不是整段作廢。

## 設計意圖

中央起點=機動空間最大、被逼到角落機動歸零——位置壓力由幾何天然提供(斜向=兩步使角落懲罰更重)。領域界線做成「可變更的判定線」而非寫死的常數,讓吃中線類機制未來能以效果形式回鍋、不動核心規則。
