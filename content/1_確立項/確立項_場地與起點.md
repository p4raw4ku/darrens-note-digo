# 確立項:場地與起點

## 已確立

- 雙方各持一個**九宮格**(3×3),起點各自中央。
- **面對面、鏡像對應**:實體為一張 3 橫排×6 縱列的連續棋盤;雙方各自視角下「我的左排=你的右排」;引擎內部用絕對座標(等價)。
- **座標系(2026-07-13 拍)**:**欄＝A–F**(縱列)、**列＝1/2/3**(橫排、上中下)——溝通與條文一律用此座標;我方 A–C、敵方 D–F。
- **跨盤距離**:沿格直數,過中線無修正;距離與範圍一律從角色所在格起算(錨點鐵則)。
- **領域界線(2026-07-13 升級為逐列)**:每一列各自一條界線、預設全在 C|D;**不可進入對方領域**(特殊效果才破例)。界線可被效果改變(現行接口=佔領詞條),變更後所有領域判定以**當前該列**界線重算。
- **佔領(界線變更詞條,2026-07-13 拍全套)**:單格版(敵前 1 格)與整欄版(敵前縱列)——①**完整欄推進鎖**:當前欄三列全歸我,才可推進下一欄(單格版用於補完未完整欄);②**站人格不變更**:佔領當下站著單位的格不變更歸屬,該單位改吃擠壓傷害 10(**直傷**,2026-07-14 批准:不經閃避/力場/護盾判定——「場地傷害」家族,掛靠見「未定與掛靠」);③敵方最後一欄(F)不可佔;④**時效=逐格回合步時鐘(2026-07-14 改制)**:**每個佔領格自己一支時鐘**(即使整欄一起佔領、各格各自計 N),自該格佔領結算完成後起算,滿 N 個回合步後退回原界線(N 暫定 12,住 [[1_準確立層]];窗要容納佔領→劍系組合兌現)。**到期時格上還有人→該格不動**,等該單位離開後的下一個回合步再退回;**退回順序由最深格往回退**——較淺格先到期則先等待、成為佔領前緣後的下一個回合步才退回(與站人等空同一個「先等待、條件解除才退」原則)。原「到期整批退回＋推人回己方」條款**作廢**(2026-07-14)。**時間口徑=回合步**:任一方行動或收手(跳過)皆算一步,與投擲時鐘同口徑(見 [[確立項_晶片]])——退回順序與回合步口徑屬實作裁量、待 user 追認;N 的換算基準因口徑變動,於連鎖動機拆解批次(層2_6)重新定錨。
- **位移硬上限**:任何位移(推/拉/自移)不得使單位移出棋盤邊界、不得使單位跨越**當前**領域界線。例:界線在 C/D 時,不能把對方從 D 拉到 C;若界線已被效果改到 B/C 之間,則 D 拉到 C 合規(對方仍在其領域內)。**逐步鉗制(2026-07-14 拍)**:多格位移逐格推進結算,碰到邊界或界線就停在最後一個合法格——**部分位移合法**、已走的格數保留,不是整段作廢。

## 設計意圖

中央起點=八方機動最大、被逼到角落機動歸零——位置壓力由幾何天然提供。領域界線做成「可變更的判定線」而非寫死的常數,讓吃中線類機制未來能以效果形式回鍋、不動核心規則。
