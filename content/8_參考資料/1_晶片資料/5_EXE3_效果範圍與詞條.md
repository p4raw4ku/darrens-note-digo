# EXE3 效果範圍與詞條系統（判讀層）

> **判讀層、非 ROM 權威**：由效果描述＋遊戲知識歸納；領航員晶片未逐張考證者在各表標「演出型(推定)」。逐張分類住各晶片表（對象／範圍／詞條欄）：[[1_EXE3_標準晶片表]]・[[2_EXE3_巨型超巨晶片表]]・[[3_EXE3_程式廣告表]]・[[4_EXE3_範本晶片組]]；本檔＝三軸系統的唯一定義處。精修＝改 `9_系統/scripts/exe3_effect_map.py` 重跑。

## 對象軸

| 對象 | 定義 |
|---|---|
| 敵單體 | 單一敵人 |
| 敵複數 | 範圍內所有敵人 |
| 敵方 | 演出型、對象隨演出 |
| 自身 | 作用於自己 |
| 我方 | 我方單位（含召喚物） |
| 指定格 | 指定格子（設置／落點） |
| 地形格 | 作用於格子本身（變地形／奪格） |
| 手牌 | 貼附在其他晶片上 |
| 特殊 | 以上皆非 |

## 範圍軸（幾何型別 → 題5 對映）

![[DIGO_EXE_範圍型別圖鑑.png]]
